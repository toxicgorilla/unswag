# unswag

## Installation

```
$ git clone <repo url>
$ cd unswag
$ bash install.sh
```

## Usage
1) Navigate to a Swagger UI page, e.g. https://website.com/swagger/index.html, and expand all the sections. 
2) Copy all the URLs ans paste them into a text file, eg. `urls.txt`
3) run `unswag urls.txt`

## Uninstallation
```
$ cd unswag
$ bash uninstall.sh
```
